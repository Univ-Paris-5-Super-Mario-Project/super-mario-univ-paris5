This game was built by four students from Paris Descartes University, as part of a University project: Xavier Lefebvre, Alexis Le Bars, Thomas Riccioli and Conrad Kleinespel.

To aid development, we used the following open source libraries:

  • JSGlib
  • AngularJS
  • UnderscoreJS
  • Buzz
  • The Mushroom Kingdom
  • Mario Universe.com
  • Twitter Bootstrap
  • Google Webfonts
